# HeartSense-Hito, Descripcion
#Las enfermedades cardiovasculares son una de las principales causas de riesgo en la población, pero a menudo los datos médicos, como los niveles de lípidos en la sangre y los signos vitales, se analizan de forma aislada.
El objetivo principal de este proyecto es estructurar esta información de manera eficiente para facilitar la detección temprana de anomalías en la salud antes de que se conviertan en emergencias, agilizando así la toma de decisiones médicas.
